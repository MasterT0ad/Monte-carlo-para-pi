# Monte-carlo-para-pi

## Introdução

O trabalho a seguir foi feito pelos estudantes João Francisco Callegari RA: 143496 e Beatriz Miranda da Silva RA: 143912 para a disciplina de Fundamentos da Computação.
O objetivo do trabalho foi utilizar o método de Monte Carlo, uma técnica computacional que se aproveita da aleatoriedade para estimar resultados que, a princípio, são incertos. Neste caso, utilizamos para estimar o valor de pi.
Para isso, utilizamos os conceitos de subprogramação e o estudo de vetores abordados em sala, junto da lógica de programação para seguir as instruções específicas dadas.

## Estrutura do projeto

O arquivo compactado enviado contém 2 arquivos:
- README com informações relevantes do projeto;
- trabalho.py com toda estrutura do código pronta para rodar
